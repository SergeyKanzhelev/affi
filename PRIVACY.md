# Privacy Policy for Affi

Affi is committed to protecting your privacy. This policy describes how the extension handles data.

## Data Collection
Affi **does not collect, store, or transmit** any personal identification information (PII). All processing happens locally within your browser.

## Data Usage
- **GitHub Content:** The extension reads the content of `OWNERS` and `OWNERS_ALIASES` files on GitHub.com to render the hierarchy and expand aliases.
- **External Requests:** The extension makes requests to `raw.githubusercontent.com` to fetch parent `OWNERS` files in the same repository.
- **Maintainer Stats:** The extension uses a locally bundled JSON file (`maintainers_stats.json`) to display contribution metrics. No user activity is sent to external servers.

## Third-Party Services
Affi interacts with GitHub's infrastructure to fetch repository configuration. Please refer to [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement) for more information.

## Changes to this Policy
We may update this policy from time to time. Any changes will be posted on the project's GitHub repository.
